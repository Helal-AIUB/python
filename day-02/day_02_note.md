# Short Note on Strings and Conditional Statements

1. **What is a string?**
   A string is a sequence of characters enclosed in single (`' '`) or double (`" "`) quotes.

2. **What is string indexing?**
   String indexing is the process of accessing individual characters in a string using their position, starting from `0`.

3. **What is string slicing?**
   String slicing is used to extract a portion of a string using the syntax `string[start:end]`.

4. **Five useful string methods:**

   * `upper()` – Converts text to uppercase.
   * `lower()` – Converts text to lowercase.
   * `strip()` – Removes leading and trailing spaces.
   * `replace()` – Replaces part of a string with another value.
   * `split()` – Splits a string into a list.

5. **What is a conditional statement?**
   A conditional statement allows a program to make decisions based on whether a condition is `True` or `False`.

6. **Difference between `if`, `elif`, and `else`:**

   * `if` checks the first condition.
   * `elif` checks another condition if the previous one is false.
   * `else` runs when none of the previous conditions are true.

7. **Difference between `and`, `or`, and `not`:**

   * `and` → True only if both conditions are true.
   * `or` → True if at least one condition is true.
   * `not` → Reverses a condition's truth value.
