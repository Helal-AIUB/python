# Short Note: Functions in Python

### 6. What is a Function?

A function is a reusable block of code that performs a specific task.

**Example:**

```python
def greet():
    print("Hello")
```

### 7. Why Should We Use Functions?

* Reduces code repetition.
* Makes programs easier to read and maintain.
* Helps organize code into smaller parts.

### 8. What is a Return Value?

A return value is the result that a function sends back using the `return` keyword.

**Example:**

```python
def add(a, b):
    return a + b
```

### 9. What is Recursion?

Recursion is a technique where a function calls itself to solve a problem.

**Example:**

```python
def countdown(n):
    if n == 0:
        return
    print(n)
    countdown(n - 1)
```
