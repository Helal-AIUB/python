6. Why is Dictionary Important for Django?

Dictionary is important in Django because it stores data in key-value pairs. Django uses dictionaries to pass data from views to templates and to handle request/response data.

7. What is a Key-Value Pair?

A key-value pair is a way of storing data where:

Key = unique identifier
Value = data associated with the key

Example:

student = {
    "name": "Rahim",
    "cgpa": 3.65
}

Here, "name" and "cgpa" are keys, while "Rahim" and 3.65 are values.

8. What is a Set?

A set is an unordered collection of unique items. It does not allow duplicate values.

Example:

numbers = {1, 2, 3, 4}

9. Write 5 List Methods
append() → Add an item at the end.
insert() → Add an item at a specific position.
remove() → Remove an item.
pop() → Remove and return an item.
sort() → Sort the list.

Example:

my_list.append(10)
10. Write 5 Dictionary Methods
keys() → Returns all keys.
values() → Returns all values.
items() → Returns key-value pairs.
get() → Returns the value of a key.
update() → Updates dictionary values.

Example:

student.get("name")
